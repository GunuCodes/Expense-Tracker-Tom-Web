# ExpenseTracker

A modern, user-friendly web application for tracking personal expenses and managing budgets.

## Features

### **Expense Tracking**
- Add, edit, and delete expenses with ease
- Categorize expenses (Food, Transportation, Entertainment, etc.)
- View detailed expense history with search and filtering
- Real-time expense summaries and statistics

### **Financial Insights**
- Interactive charts and graphs showing spending patterns
- Category breakdown with visual representations
- Monthly budget progress tracking
- Recent transactions overview

### **Budget Management**
- Set monthly budget goals
- Track spending against budget limits
- Visual progress indicators
- Budget alerts and notifications

### **User Account**
- Secure user registration and login
- Google OAuth integration for quick sign-in
- Personal profile management
- Customizable settings and preferences

### **User Experience**
- Clean, modern interface with dark mode support
- Fully responsive design for all devices
- Intuitive navigation and user-friendly forms
- Fast, reliable performance

### **Security & Privacy**
- Secure authentication with JWT tokens
- Password hashing and data encryption
- Privacy policy and terms of service
- Admin panel for system management

Built with:
- **Frontend**: Vanilla JavaScript, HTML, CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT + Google OAuth
- **Charts**: Chart.js for data visualization
