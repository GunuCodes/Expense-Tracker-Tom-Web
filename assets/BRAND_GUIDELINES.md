# ExpenseTracker Brand Guidelines

## Color Palette

### Primary Colors
- **Primary Dark Navy**: `#084298` (--color-primary-700)
- **Primary Darkest**: `#031633` (--color-primary-900)
- **Primary Base**: `#0d6efd` (--color-primary-500)

### Secondary Colors
- **Secondary Teal**: `#14b8a6` (--color-secondary-500)
- **Secondary Dark**: `#0f766e` (--color-secondary-700)

### Accent Colors
- **Accent Purple**: `#7c3aed` (--color-accent-600)

## Typography

### Font Families
- **Headings**: Poppins (400, 500, 600, 700, 800)
- **Body**: Inter (300, 400, 500, 600, 700, 800)

### Logo Typography
- Font: Poppins
- Weight: 700 (Bold)
- "Expense" in Primary Dark Navy (#084298)
- "Tracker" in Secondary Teal (#14b8a6)

## Logo Usage

### Full Logo
- Use `assets/logos/logo.svg` for full logo
- Minimum size: 150px width
- Maintain aspect ratio (4:1)

### Icon Only
- Use `assets/logos/favicon.svg` for icon-only
- Size: 32x32px for favicon
- Size: 192x192px and 512x512px for app icons

### Logo Placement
- Header: Left-aligned, 200px width
- Footer: Centered, 150px width
- Favicon: 32x32px in browser tab

## Icon System

### Icon Library
- **Font Awesome 6.5.1** (Free version)
- Style: Solid and Regular

### Icon Colors
- Primary actions: Primary color (#084298)
- Secondary actions: Secondary color (#14b8a6)
- Neutral: Gray-600 (#4b5563)
- Success: Green-600 (#16a34a)
- Error: Red-600 (#dc2626)

### Icon Sizes
- Small: 16px (0.875rem)
- Base: 20px (1rem)
- Large: 24px (1.5rem)
- Extra Large: 32px (2rem)

## Image Guidelines

### Hero Images
- Aspect ratio: 16:9
- Format: WebP or JPG
- Max width: 1200px
- Border radius: 16px
- Shadow: --shadow-lg

### Illustrations
- Style: Modern, minimalist
- Colors: Match brand palette
- Format: SVG preferred, PNG fallback
- Border radius: 12px

### Avatars
- Size: 40x40px (default)
- Shape: Circle
- Border: 2px solid --color-border-light
- Shadow: --shadow-sm

## Spacing & Layout

### Consistent Spacing
- Use 4px base unit system
- Standard gaps: 8px, 16px, 24px, 32px

### Image Containers
- Padding: 16px (--spacing-4)
- Margin: 24px (--spacing-6) between images
- Max container width: 1200px

## Do's and Don'ts

### ✅ Do
- Use brand colors consistently
- Maintain logo aspect ratio
- Use SVG for logos and icons when possible
- Ensure proper contrast ratios (WCAG AA)
- Keep images optimized (< 200KB)

### ❌ Don't
- Don't stretch or distort logos
- Don't use colors outside the palette
- Don't use low-resolution images
- Don't place logos on busy backgrounds
- Don't modify logo colors (except monochrome versions)
